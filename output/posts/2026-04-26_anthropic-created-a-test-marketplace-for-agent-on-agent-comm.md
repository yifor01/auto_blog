📌 Anthropic 內部試辦 Agent 對 Agent 交易市集

AI Agent 幫你買東西砍價，你以為拿到最優結果？Anthropic 內部實驗發現，用更強模型的用戶交易成果更好，但當事人完全沒察覺差異，甚至可能吃虧還不知道。

🤔 **Agent 間自動商務實測，呼應 Agent 經濟發展趨勢**
這項試驗名為 Project Deal，是 Anthropic 針對 Agent 對 Agent 自動交易場景的初步探索，呼應當前 Agent 經濟的發展趨勢。團隊搭建了一個 AI Agent 同時擔任買家與賣家的分類市集，測試真實場景下 Agent 自主完成交易的可行性。

🧪 **69 名自選員工參與，4 個市集對照設計**
試驗招募 69 名 Anthropic 員工組成自選參與池，每人獲得 100 美元預算（以禮品卡形式發放），用於購買同事的物品。團隊共運行 4 個獨立市集：1 個「真實」市集採用 Anthropic 最先進模型代表用戶，交易在實驗結束後實際兌現；其餘 3 個為研究用途市集。最終共完成 186 筆交易，總價值超過 4000 美元。

💡 **更強模型成果更好，用戶無感 Agent 質量落差**
實驗結果顯示，由更先進 AI 模型代表的用戶，客觀上能獲得更好的交易結果。但參與用戶並未察覺這種差距，這意味著可能出現「Agent 質量落差」：使用較弱模型的用戶處於劣勢，卻完全不知道自己更吃虧。
另外團隊發現，給予 Agent 的初始指令，並不會影響成交機率或議價後的價格。

⚠️ **小樣本內部試點，尚無工程落地價值**
本次試驗屬於初步探索性質，存在明顯限制：參與者僅為 Anthropic 內部員工，屬於自選樣本，規模小且缺乏代表性；未公開技術細節、無開源程式碼，也沒有可對外提供的工具或 API。對工程師而言，目前尚無可直接落地的應用價值。
