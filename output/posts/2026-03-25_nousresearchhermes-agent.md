📌 【NousResearch】Hermes Agent：$5 VPS 也能自我進化的 AI Agent  

🎣 你以為只有雲端巨廠才能打造會自我學習的 AI？Hermes 證明只要一台低成本 VPS 就能擁有閉環學習迴圈，隨時隨地透過 Telegram、Discord 或 CLI 與它對話。  

🤔 **為什麼這種「自改進」代理值得關注**  
現有的 AI agent 框架多依賴外部微調或人工干預來提升能力。NousResearch 的 Hermes Agent 提出一種內建學習迴圈：它會從使用經驗中創造技能，在實際運作過程中持續改進，並透過定期提醒將知識固化。這意味著代理不只是被動執行指令，而是能根據過去的互動自行強化自身行為模式。  🧪 **核心功能與設計亮點**  
- **跨平台單一閘道**：同一支程式可同時接入 Telegram、Discord、Slack、WhatsApp、Signal 與命令列，無需為每個平台維護獨立客戶端。  
- **模型自由切換**：支援 Nous Portal、OpenRouter（200+ 模型）、z.ai/GLM、Kimi/Moonshot、MiniMax、OpenAI 或自訂端點，透過 `hermes model` 指令即時切換，零程式碼更動、無供應商鎖定。  
- **完整終端介面 (TUI)**：多行編輯、斜線指令自動補全、對話歷史、中斷與重導向、串流工具輸出，讓終端使用體驗接近傳統 Shell。  - **語音與跨平台對話延續**：支援語音備忘錄轉文字，並能在不同裝置與平台間保持對話上下文。  
- **閉環學習機制**：  
  - Agent‑curated 記憶配合定期 nudges，促使知識長期保留。  
  - 複雜任務完成後自主創造新技能，且該技能在使用過程中自我優化。  
  - 使用 FTS5 全文搜尋搭配 LLM 摘要，實現跨會話的記憶檢索與回顧。  
  - 採用 Honcho 辯證式使用者建模，逐步建構使用者個人模型。  
- **自動化與排程**：內建 cron 排程器，可將任務結果發送至任意平台；日常報告、夜間備份、週次審核均以自然語言呈現，可完全無人看管運行。  - **開放標準相容**：符合 agentskills.io 開放標準，便於與其他工具或服務互操作。
