## 🧪 最新加入的實用工具

### 🔬 Claude Scientific Skills by K-Dense
一個準備好使用的 Agent Skills 集合，涵蓋研究、科學、工程、分析、財務和寫作等領域。簡單描述卻功能強大，是 GitHub 上最優秀的 Skills 資源庫之一。

### 🛡️ parry by Dmytro Onypko
針對 Claude Code hooks 的 prompt injection 掃描工具，檢查工具輸入輸出中的注入攻擊、秘密資訊和數據外洩嘗試。雖然還在早期開發階段，但值得關注。

### ✅ Dippy by Lily Dayton
使用 AST 基礎解析自動批准安全的 bash 命令，同時對破壞性操作提出確認提示。解決了許可疲勞問題，同時保持安全性。支援 Claude Code、Gemini CLI 和 Cursor。

### 🛠️ sudocode by ssh-randy
輕量級代理協調開發工具，直接存在於你的儲存庫中。整合多種規格框架，讓專案管理更像 Jira。

### 🖥️ claude-tmux by Niels Groeneveld
在 tmux 中管理 Claude Code。提供所有 Claude Code 實例的 tmux popup，支援快速切換、狀態監控、會話生命週期管理，並整合 git worktree 和 pull request 支援。
