📌 **OpenCode：開源 AI 編碼助手**  你是否厭倦了付費的 AI 編程訂閱？一個號稱完全開源、不儲存你程式碼的新工具正在 Hacker News 上爆紅。  

🤔 **付費訂閱不是必備，開源模式降低使用門檻**  
隨著 GitHub Copilot、Codeium 等商業輔助工具成為開發者的標配，許多團隊開始評估長期成本與資料隱私。OpenCode 的出現提供了一個可以直接連結任意模型（含 Claude、GPT、Gemini 及本地模型）且不需額外訂閱的替代方案。  

🧪 **跨平台桌面、終端機與 IDE 三種介面，內建 LSP 多會話支援**  
OpenCode 提供 macOS、Windows、Linux 桌面應用程式的 beta 版，同時可作為終端機指令或 IDE 擴充功能使用。其主要特色包括：  
- 自動載入適當的 LSP 供大型語言模型使用  
- 支援在同一專案中啟動多個代理會話（multi‑session）  
- 可分享會話連線以供參考或除錯  
- 透過 GitHub 帳號登入使用 Copilot 權限  
- 透過 OpenAI 帳號登入使用 ChatGPT Plus/Pro  
- 透過 Models.dev 接入超過 75 個 LLM 提供者，包含本地模型  
- 不儲存任何使用者程式碼或上下文資料，強調隱私優先
