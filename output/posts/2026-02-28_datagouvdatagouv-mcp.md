📌 **法國政府開放資料，現在可以用對話取得！**

你還在為了找政府開放資料，而一個個點開網頁、下載 CSV、再手動分析嗎？現在，透過 MCP (Model Context Protocol) 協議，你可以直接對話取得法國 data.gouv.fr 平台上的所有開放資料！

🤔 **為什麼這很重要？**

data.gouv.fr 是法國國家級的開放資料平台，收錄了政府部門、公共機構、地方政府等提供的各類資料。過去要使用這些資料，開發者必須：
- 手動搜尋合適的資料集
- 下載並解析各種格式的文件
- 自行整合不同來源的資料

現在，透過 MCP server，AI 助手可以直接存取這個資料庫，讓資料搜尋和分析變得像聊天一樣簡單。

🧪 **怎麼使用？**

最簡單的方法是使用公共端點：
```
https://mcp.data.gouv.fr/mcp
```
完全免費，無需認證！

支援的聊天工具包括：
- Claude Code
- ChatGPT
- Gemini CLI
- VS Code
- Cursor
- 以及許多其他 MCP 客戶端

 **實際範例**

你可以直接問：
- "Quels jeux de données sont disponibles sur les prix de l'immobilier ?" (哪些資料集有房地產價格？)
- "Montre-moi les dernières données de population pour Paris" (給我巴黎最新的人口資料)
- "Trouve-moi des données sur l'énergie renouvelable en France" (找法國可再生能源資料)

💡 **技術亮點**

這不只是個簡單的 API wrapper。MCP server 讓 AI 助手能：
- 理解你的自然語言查詢
- 搜尋相關的資料集
- 提取和分析資料內容
- 直接在對話中呈現結果

🎯 **實務應用**

- 研究人員可以快速取得政策相關數據
- 開發者能快速原型化資料驅動的應用
- 記者可以即時查證官方數據
- 學生可以更容易地取得學習資料

🔗 **開始使用**

1. 選擇你的 AI 助手（推薦 Claude Code 或 ChatGPT）
2. 設定 MCP 伺服器為 `https://mcp.data.gouv.fr/mcp`
3. 開始用自然語言提問！
