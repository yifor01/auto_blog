📌 【微軟 Agent Framework】跨語言 AI 代理框架正式開源，圖形化工作流 + 完整工具鏈

微軟正式開源了 Agent Framework，一個跨語言的 AI 代理框架，支援 .NET 和 Python 兩種實作，旨在簡化 AI 代理的構建、協調和部署。

🤔 **企業級 AI 應用的新選擇**

隨著 AI 代理技術的快速發展，開發者面臨的挑戰是如何從零開始構建可靠的代理系統。Agent Framework 提供了一套完整的工具鏈，從簡單的對話代理到複雜的多代理工作流，都能快速實現。

🧪 **核心特色搶先看**

- **圖形化工作流**：基於圖表的代理和確定性函數連接，支援串流、檢查點、人機介入和時間旅行功能
- **雙語言支援**：Python 和 .NET 實作，滿足不同技術棧的需求
- **實驗性功能**：AF Labs 包含基準測試、強化學習等前沿研究功能
- **開發者體驗**：DevUI 提供互動式開發界面，加速開發流程

🎣 **安裝與快速上手**

Python 使用者只需執行：
```bash
pip install agent-framework --pre
```

.NET 使用者則執行：
```bash
dotnet add package Microsoft.Agents.AI
```

完整的文件和教程可幫助開發者快速入門。
